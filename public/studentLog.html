ENTRANCE

Scan a student's ID QR Code and check if it is validated before allowing entry to the campus. If the scan results an error, it means that the student's ID is not yet validated.

EXIT

Scan a student's ID QR Code and check if the student has entered the campus using that ID. If the scan results an error, it means that the student didn't enter the campus using their ID.
